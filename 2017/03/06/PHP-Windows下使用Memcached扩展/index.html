PHP Windows下使用Memcached扩展

发表于 2017-03-06 | 分类于 PHP

github上发现了一个很好用的小文件，能够
模拟出php_memcached.dll的PHP memcached扩展。

PHP memcached client

已知PCEL有两个memcache client的扩展，一个是memcache，一个是memcached，他们的实现方式不同，后者更推荐使用。区别自行查找资料。

假定你的生产环境使用的是memcached扩展，而不是memcache扩展

我的Windows开发环境用的是XAMPP，已经包含了PCEL的 memcache.dll扩展，但是我希望使用基于libmemcached开发的memcached PCEL扩展来连接我的memcached server。但由于找不到Windows平台的memcached 扩展。可以使用该memcached的模拟文件。这样就可以直接使用，并且可以发布到装了memcached扩展的Linux生产环境里而不需要修改代码。

在使用该文件之前，只能这么写：

# windows # memcached # php
